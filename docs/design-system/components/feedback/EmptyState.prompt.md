Dashed-outline placeholder for empty lists — "Aún no hay lotes" and equivalents in /library, /gallery, /personas. Composes `Button` for the primary action.

```jsx
<EmptyState title="Aún no hay lotes" description="Pega una URL de producto o escribe una descripción para lanzar tu primer lote." actionLabel="Nuevo lote" onAction={createBatch} />
```
